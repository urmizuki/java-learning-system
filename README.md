# Java Fundamentals

## 📌 Overview
This project contains my practice and implementation of core Java concepts including OOP and basic programming exercises.

## 🧠 Topics Covered
- Object-Oriented Programming (Classes, Objects)
- Encapsulation
- Basic data structures practice
- Java fundamentals exercises

## 📁 Structure
- `src/oop` → OOP-based lab implementations
- `src/exercises` → practice and small coding exercises
- `Main.java` → entry point

## 🚀 Purpose
To strengthen my understanding of core Java before moving into advanced software engineering and backend development.

## ▶️ How to run
Compile and run `Main.java` in any Java IDE or terminal.
